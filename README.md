## О Программе
**Arcanit**

Программа выполняет поиск пользователей и репозиториев на  **https://api.github.com**

Пока идет процесс загрузки, поле для ввода и кнопка поиска неактивны.
Пока длина строки для поиска менее трёх символов, кнопка поиска неактивна.

При нажатии на пользователя осуществляется загрузка его страницы в браузере. 
Если браузеров на устройстве несколько, предлагается выбрать из установленных.

При нажатии на репозиторий открывается его содержимое **файлы** и **папки**.
При открытии папки открывается webView с её содержимым.
При открытии папки открывается её содержимое.
